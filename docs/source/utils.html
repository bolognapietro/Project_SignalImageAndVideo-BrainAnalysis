utils module

Provides basic functions related to images.

utils.load_img(filename: str) → ndarray

Loads an image.

Parameters:

filename (str) – Source file.

Returns:

Loaded image.

Return type:

np.ndarray

utils.save_img(src: ndarray, filename: str, overwrite: bool = True) → bool

Saves an image.

Parameters:

  • src (np.ndarray) – Image to be saved.

  • filename (str) – Destination file.

  • overwrite (bool) – Optional, if true, the specified destination file will be overwritten if it exists. Defaults to true.

Returns:

true on success, false otherwise. If the destination file exists and overwrite is set on false, the function will return false.

Return type:

bool

utils.show_img(src: ndarray | list, winname: str = 'Image', ncolumns: int = 3) → None

Shows an image or an array of images.

Parameters:

  • src (np.ndarray or list[np.ndarray]) – Image(s) to be displayed.

  • winname (str) – Optional, window name. Defaults to “Image”.

  • ncolumns (int) – Optional, number of columns if multiple images are displayed. It’s ignored if src param is a single image. Defaults to 3.

Returns:

None

Return type:

None
